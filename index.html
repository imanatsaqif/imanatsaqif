Wastify

Wastify stands as a pioneering mobile application designed to revolutionize waste management through cutting-edge machine learning. The app empowers users to effortlessly identify diverse types of waste, offering insights into proper disposal methods and locating nearby waste collection points.

My Role: Machine Learning Engineer

  • TensorFlow proficiency for model development
  • Python programming for end-to-end implementation
  • Mastery of image data processing using Keras' ImageDataGenerator
  • Leveraging transfer learning with Xception model for accuracy
  • Neural network architecture design
  • Meticulous model evaluation and performance optimization
  • Early stopping techniques implementation
  • Docs
  • GitHub

Swipets

Swipets is a groundbreaking pet dating app that seamlessly bridges pet lovers' desire for companionship with modern technology. The app operates similarly to a conventional dating app, but profiles, pictures, and descriptions represent the owner's pets. Its purpose is to facilitate pet owners in finding healthy partners for breeding.

My Role: UI/UX Designer and Researcher

  • Figma proficiency for UI design and prototyping
  • Mobile app design for seamless user experience
  • User experience (UX) research for informed design decisions
  • Prototyping for demonstrating app flow and functionality
  • Wireframing to outline app layout and structure
  • Visual design with graphics, icons, and colors
  • Collaborative design for effective team feedback
  • Responsive design for adaptability to various screens
  • Usability testing for validation and user-centric design
  • Docs
